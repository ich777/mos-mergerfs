# MOS mergerfs

MOS mergerfs provides **prebuilt and packaged mergerfs binaries** for use within the MOS ecosystem.

This repository contains the **build scripts, packaging logic, and configuration files** required to compile and package mergerfs for MOS.

---

## Overview

mergerfs is obtained from **upstream prebuilt releases** and packaged for use in MOS.

No functional changes to mergerfs itself are intended.  
The goal of this repository is to provide **consistent and reproducible mergerfs packages** for MOS installations.
